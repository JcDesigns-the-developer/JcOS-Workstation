## 🚀 Overview

**JcOS Workstation** is a custom Windows ISO designed for developers who want a ready-to-code system immediately after install.

No manual setup. No wasted time.
Everything is preconfigured for modern development workflows.

## 👨‍💻 Built For

* Software Developers
* Embedded / Hardware Developers
* Scripters & Automation Users
* DevOps / Backend Workflows

## 🔥 Features

* 🐧 WSL (Linux environment ready)
* ⚙️ Automated first-boot setup
* 🧰 Preinstalled development tools
* 🔌 Built-in ESP32 / Arduino driver support
* 🖥 Clean, debloated Windows environment
* ⚡ Fast boot + optimized performance
* 🔒 Controlled updates (no forced updates)
